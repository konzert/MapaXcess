# gobz
### Run the backend
Move to backend folder:
```bash
cd gobz-backend
```
Install all dependencies:
```bash
yarn install
```
Run the backend:
```bash
yarn start
```

| :exclamation:  Don't use npm please   |
|-----------------------------------------|
### Run applications in Docker development
Build the image:
```bash
docker-compose build
```
Start the containers:
```bash
docker-compose up
```
Stop the containers:
```bash
docker-compose down
```
